# Hunggg NewTab

Free to use.

Custom Chrome New Tab extension (Manifest V3) with:
- Video wallpaper + poster fallback
- Clock + greeting
- Search box
- Weather widget
- Music widget
- Quick links (Facebook, YouTube, TikTok, Reddit)
- Add custom links (+)
- GPU Boost toggle (hide video, use poster)

## Run / Activate in Chrome
1. Open `chrome://extensions`.
2. Enable **Developer mode** (top-right).
3. Click **Load unpacked**.
4. Select this project folder.
5. Open a new tab to use it.

## Reload after updates
1. Go to `chrome://extensions`.
2. Find `Hunggg NewTab`.
3. Click **Reload**.
